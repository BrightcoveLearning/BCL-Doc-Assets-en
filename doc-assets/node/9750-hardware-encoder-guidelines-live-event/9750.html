The following is a list of encoders that are generally known to work with Video Cloud. Other encoders may also work; this list contains the encoders that we have specifically tested.

Also note that supported does not mean that every feature is supported. Specific capabilities (such as sending cuepoints) vary among different encoders.
  • XSplit
  • OBS
  • Wirecast
  • AWS Elemental Live
  • Haivision
  • LiveU
  • Matrox
  • NewTek
  • Teradek
